# Fundoo
  Fundoo is Todo Notes Application inspired from Keep. It is like clone of keep.

## Start With Dev Server
```
npm start
```

## Initiate Dev Build
```
npm run dev
```
Build Location: fundoo/dev


## Initiate Production Build
```
npm run prod
```
Build Location: fundoo/prod


## Starting Dev Build
```
npm run start-dev
``` 

## Starting Production Build
```
npm run start-prod
```
